## How to Install African-Dawn?

0. Unpack the contents of the archive somewhere.
1. Copy the folder "afrika_dawn", file "generate_path.bat" and the file "afrika_dawn.mod" to the folder %UserProfile%/Documents/Paradox Interactive/Hearts of Iron IV/mods (copy and paste this path into the file explorer).
2.1 Run the file "generate_path.bat"

OR

2.2 Edit the file "afrika_dawn.mod", specifying the FULL path to the folder "afrika_dawn" (Example: path="C:/Users/sepera_okeq/Desktop/test/afrika_dawn").
